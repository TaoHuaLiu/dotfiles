# Installation

- Put `~/.local/bin` in you `$PATH`
- `git clone --bare`

# Programs 

- Neovim
    - Oriented around pandoc
    - Plugins with vim-plug
- Newsboat
- Zsh
    - vi-mode
    - autojump
    - zsh-syntax-highlighting
    - zsh-autosuggestions
- lf
    - Icons
- AwesomeWM
    - dmenu laucher
- Zathura
    - Arc Theme
    - Without GUI
- Alacritty
    - mononoki font
    - No window borders
- Xinit
    - Redshift
    - remaps for a compose key (left meta) and a caps lock=control
    - Hi-DPI

# Scripts

- remaps: set using `setxkbmap`
